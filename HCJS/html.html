🔹 一、基础结构标签

作用：定义 HTML 文档的基本骨架，告诉浏览器文档类型和结构。

1. <!DOCTYPE html>

作用：声明文档类型为 HTML5，让浏览器以标准模式渲染页面。

<!DOCTYPE html>

2. <html>

作用：HTML 文档的根标签，所有其他标签都嵌套在其中。

<html lang="zh-CN">
  
</html>

3. <head>

作用：包含页面的元数据（如字符编码、标题、样式等），不会在页面中显示。

<head>
  <meta charset="UTF-8">
  <title>页面标题</title>
</head>

4. <body>

作用：包含所有可见的页面内容，如文本、图片、链接等。

这是 <body> 标签内的内容，会在浏览器中显示。

5. <meta> 扩展属性

作用：设置页面视口、描述、关键词等，提升SEO和移动端适配。

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="HTML核心知识点解析，覆盖日常开发常用标签">
<meta name="keywords" content="HTML,前端开发,HTML标签,语义化">

6. <base> 标签

作用：设置页面中所有相对链接的基准URL，简化链接书写。

<base href="https://www.example.com/" target="_blank">

🔹 三、链接与导航

作用：创建页面内跳转、外部链接、下载链接等。

1. 链接标签 <a>

核心属性：href（目标地址）、target（打开方式，如 _blank 新窗口打开）、download（下载文件）。

打开百度（新窗口） | 跳转到「基础结构标签」 | 下载文件

2. 特殊用途链接

作用：创建邮件、电话链接，设置链接安全/SEO属性。

发送邮件 | 拨打电话 | 安全外链（防钓鱼） 

<a href="mailto:contact@example.com?subject=咨询&body=我想了解HTML相关内容">发送邮件（带主题/内容）</a>

<a href="https://example.com" rel="nofollow">nofollow（SEO不传递权重）</a>

🔹 四、图片与多媒体

作用：在页面中插入图片、视频、音频或嵌入其他页面。

1. 图片标签 <img>

核心属性：src（图片地址）、alt（图片加载失败时的替代文本）、width/height（尺寸）。

（图片加载失败时会显示 alt 文本：示例图片）

<img src="https://picsum.photos/300/200" alt="示例图片" width="300" height="200">

2. 内联框架 <iframe>

作用：在当前页面嵌入另一个网页（如地图、视频等）。

<iframe src="https://www.baidu.com" width="100%" height="300" frameborder="0"></iframe>

3. 视频标签 <video>

作用：嵌入视频文件，核心属性：src（视频地址）、controls（控制栏）、autoplay（自动播放）、loop（循环）、muted（静音）。

<video src="video.mp4" controls autoplay loop muted width="300">
  您的浏览器不支持 video 标签，请升级浏览器
</video>

<video controls width="300">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  您的浏览器不支持 video 标签
</video>

4. 音频标签 <audio>

作用：嵌入音频文件，核心属性同video（无width/height）。

<audio src="music.mp3" controls autoplay loop muted>
  您的浏览器不支持 audio 标签
</audio>

<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
  您的浏览器不支持 audio 标签
</audio>

5. 响应式图片 <picture>

作用：根据屏幕尺寸/设备类型加载不同图片，提升性能。

<picture>
  <source media="(min-width: 1200px)" srcset="large.jpg">
  <source media="(min-width: 768px)" srcset="medium.jpg">
  <source media="(max-width: 767px)" srcset="small.jpg">
  <img src="fallback.jpg" alt="响应式图片">
</picture>

🔹 七、表单标签

作用：收集用户输入信息，实现与用户的交互（如登录、注册、搜索）。

1. 表单容器 <form>

核心属性：action（提交地址）、method（提交方式：GET/POST）、enctype（编码类型）。

<form action="/api/login" method="POST" enctype="application/x-www-form-urlencoded">
  <!-- 表单控件 -->
  <button type="submit">提交</button>
  <button type="reset">重置</button>
</form>

2. 常用表单控件

作用：不同类型的输入控件，满足不同的输入需求。

<!-- 单行文本 -->
<input type="text" name="username" placeholder="提示文字" required maxlength="20">
<!-- 密码框 -->
<input type="password" name="pwd" autocomplete="off">
<!-- 单选框（name相同为一组） -->
<input type="radio" name="gender" value="male" checked> 男
<input type="radio" name="gender" value="female"> 女
<!-- 复选框 -->
<input type="checkbox" name="hobby" value="game" checked> 游戏
<!-- 下拉框 -->
<select name="level">
  <option value="" disabled selected>请选择等级</option>
  <option value="1">初级</option>
  <option value="2">中级</option>
</select>
<!-- 多行文本 -->
<textarea name="desc" rows="4" cols="50"></textarea>
<!-- 文件上传（多文件加multiple） -->
<input type="file" name="avatar" accept="image/*" multiple>
<!-- 按钮类型 -->
<button type="button">普通按钮</button>
<button type="submit">提交按钮</button>
<button type="reset">重置按钮</button>

3. 表单验证与优化

作用：通过原生属性实现基础验证，提升用户体验。

<!-- 必填项 -->
<input type="text" required>
<!-- 格式验证（邮箱/手机号） -->
<input type="email" required>
<input type="tel" pattern="^1[3-9]\d{9}$" title="请输入正确的手机号">
<!-- 长度限制 -->
<input type="text" minlength="6" maxlength="18">
<!-- 禁用/只读 -->
<input type="text" disabled>
<input type="text" readonly value="固定内容">

🔹 八、语义化布局标签

作用：用有意义的标签替代<div>，提升代码可读性、SEO友好性。

1. 核心语义化标签

header（页面/区块头部）

包含标题、logo、导航等

nav（导航区域）

页面主要导航链接

main（页面主内容）

section（主题区块）

相关内容的集合（如文章列表、商品分类）

article（独立文章）

可独立分发的内容（如博客、新闻）

aside（侧边栏）

辅助内容（如广告、推荐、作者信息）

footer（页面/区块底部）

包含版权、联系方式、备案信息等

<!-- 完整语义化布局示例 -->
<header>
  <h1>网站标题</h1>
  <nav>导航链接</nav>
</header>

<main>
  <section>
    <h2>文章列表</h2>
    <article>文章1</article>
    <article>文章2</article>
  </section>
  <aside>侧边栏推荐</aside>
</main>

<footer>
  <p>© 2025 版权所有</p>
</footer>

2. 其他语义化标签

mark（标记文本） | 2025-01-01（time：时间标签） |

address（联系地址）：北京市朝阳区XX路XX号 

<!-- 标记文本 -->

<!-- 时间标签（机器可读） -->
<time datetime="2025-01-01T12:00:00+08:00">2025年1月1日 中午12点</time>
<!-- 联系地址 -->
<address>
  作者：张三 <br>
  邮箱：zhangsan@example.com
</address>
<!-- 进度/度量 -->
<progress value="70" max="100">70%</progress>
<meter value="80" min="0" max="100" low="30" high="80">80分</meter>

  • ✅ 代码可读性更高：一眼识别标签用途
  • ✅ SEO更友好：搜索引擎更易理解内容结构
  • ✅ 无障碍访问：屏幕阅读器可正确解析内容
  • ✅ 维护成本更低：团队协作更易理解代码
