🔹 一、CSS基础入门

1. CSS是什么？

CSS（Cascading Style Sheets）层叠样式表，用于描述HTML文档的呈现样式，控制网页的布局、颜色、字体等视觉效果。

2. CSS引入方式

1. 行内样式（内联样式） 
<div style="color: red; font-size: 16px;">行内样式</div>
2. 内部样式表 
<head>
  <style>
    div {
      color: red;
      font-size: 16px;
    }
  </style>
</head>
3. 外部样式表（推荐） 
<!-- HTML文件中引入 -->
<link rel="stylesheet" href="style.css">

<!-- style.css文件内容 -->
div {
  color: red;
  font-size: 16px;
}

3. CSS语法规则

/* 选择器 { 属性: 值; 属性: 值; } */
/* 基础语法示例 */
selector {
  property: value;
  another-property: value;
}

/* 实际示例 */
p {
  color: #333;
  font-size: 14px;
  line-height: 1.6;
}

🔹 二、CSS选择器详解

选择器用于选择需要设置样式的HTML元素，是CSS的核心基础。

1. 基础选择器

选择器类型 语法 说明 示例
元素选择器 标签名 选择所有指定标签 p { color: red; }
类选择器 .类名 选择所有指定类的元素 .box { border: 1px solid #000; }
ID选择器 #ID名 选择唯一指定ID的元素 #header { background: #f5f5f5; }
通配符选择器 * 选择所有元素 * { margin: 0; padding: 0; }

2. 复合选择器

选择器类型 语法 说明 示例
后代选择器 选择器1 选择器2 选择选择器1内部的所有选择器2元素 .nav li { list-style: none; }
子元素选择器 选择器1 > 选择器2 选择选择器1直接子元素的选择器2 ul > li { margin: 5px 0; }
并集选择器 选择器1, 选择器2 同时选择多个选择器的元素 h1, h2, h3 { color: #333; }
交集选择器 选择器1选择器2 选择同时满足多个条件的元素 p.active { font-weight: bold; }
相邻兄弟选择器 选择器1 + 选择器2 选择紧跟在选择器1后的选择器2元素 h2 + p { margin-top: 10px; }
通用兄弟选择器 选择器1 ~ 选择器2 选择选择器1之后的所有选择器2元素 h2 ~ p { color: #666; }

3. 属性选择器

/* 选择有href属性的a标签 */
a[href] { color: #007bff; }

/* 选择href值等于指定值的a标签 */
a[href="https://www.baidu.com"] { color: #dc3545; }

/* 选择href值包含指定字符串的a标签 */
a[href*="example"] { text-decoration: none; }

/* 选择href值以指定字符串开头的a标签 */
a[href^="http"] { background: url(icon-external.png) no-repeat; }

/* 选择href值以指定字符串结尾的a标签 */
a[href$=".pdf"] { background: url(icon-pdf.png) no-repeat; }

4. 伪类选择器

常用伪类： 
/* 链接伪类 */
a:link { color: #007bff; }        /* 未访问的链接 */
a:visited { color: #6c757d; }    /* 已访问的链接 */
a:hover { color: #dc3545; }      /* 鼠标悬停的链接 */
a:active { color: #ffc107; }     /* 鼠标点击的链接 */

/* 表单伪类 */
input:focus { border-color: #007bff; outline: none; }  /* 获取焦点 */
input:disabled { background: #e9ecef; }               /* 禁用状态 */
input:checked { accent-color: #007bff; }              /* 选中状态 */

/* 结构伪类 */
li:first-child { border-top: 1px solid #eee; }        /* 第一个子元素 */
li:last-child { border-bottom: 1px solid #eee; }      /* 最后一个子元素 */
li:nth-child(2) { background: #f8f9fa; }              /* 第二个子元素 */
li:nth-child(even) { background: #f8f9fa; }           /* 偶数行 */
li:nth-child(odd) { background: #ffffff; }            /* 奇数行 */

/* 否定伪类 */
p:not(.special) { color: #666; }                      /* 排除指定类的元素 */

5. 伪元素选择器

/* 常用伪元素 */
p::first-letter { font-size: 24px; font-weight: bold; }  /* 第一个字符 */
p::first-line { color: #dc3545; }                         /* 第一行文本 */
div::before { content: "前缀："; color: #007bff; }        /* 元素前插入内容 */
div::after { content: "后缀"; color: #dc3545; }           /* 元素后插入内容 */
::selection { background: #ffc107; color: #333; }         /* 选中的文本 */

/* 清除浮动伪元素 */
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

6. 选择器优先级

优先级从高到低：
!important > 行内样式 > ID选择器 > 类选择器/伪类/属性选择器 > 元素选择器/伪元素 > 通配符 > 继承的样式
计算规则： 内联样式(1,0,0,0) > ID选择器(0,1,0,0) > 类选择器(0,0,1,0) > 元素选择器(0,0,0,1)

🔹 三、CSS盒模型

CSS盒模型是CSS布局的基础，每个HTML元素都可以看作一个矩形的盒子。

1. 盒模型组成

  • content（内容区）： 元素的实际内容，如文本、图片等
  • padding（内边距）： 内容区与边框之间的空白区域
  • border（边框）： 内边距外的线条
  • margin（外边距）： 边框外与其他元素之间的空白区域

2. 盒模型类型

/* 标准盒模型（W3C盒模型）- 默认 */
.box {
  box-sizing: content-box;
  width: 200px;    /* 宽度仅包含content */
  padding: 20px;
  border: 5px solid #000;
  /* 实际宽度 = 200 + 20*2 + 5*2 = 250px */
}

/* IE盒模型（怪异盒模型）- 推荐使用 */
.box {
  box-sizing: border-box;
  width: 200px;    /* 宽度包含content+padding+border */
  padding: 20px;
  border: 5px solid #000;
  /* 实际宽度 = 200px（content自动调整为150px） */
}

/* 全局设置border-box（开发常用） */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

3. 常用盒模型属性

1. 内边距（padding）： 
/* 简写方式 */
padding: 10px;                /* 四个方向都是10px */
padding: 10px 20px;           /* 上下10px，左右20px */
padding: 10px 20px 30px;      /* 上10px，左右20px，下30px */
padding: 10px 20px 30px 40px; /* 上、右、下、左（顺时针） */

/* 单独设置 */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;
2. 边框（border）： 
/* 简写方式 */
border: 1px solid #000;       /* 宽度 样式 颜色 */

/* 单独设置 */
border-width: 1px;            /* 边框宽度 */
border-style: solid;          /* 边框样式：solid/dashed/dotted/double等 */
border-color: #000;           /* 边框颜色 */

/* 单独设置某一边 */
border-top: 1px solid #000;
border-right: 2px dashed #f00;
border-bottom: 3px dotted #0f0;
border-left: 4px double #00f;

/* 圆角边框 */
border-radius: 4px;           /* 四个角都是4px */
border-radius: 4px 8px;       /* 左上/右下 4px，右上/左下 8px */
border-radius: 50%;           /* 圆形（宽高相等时） */
3. 外边距（margin）： 
/* 简写方式（同padding） */
margin: 10px;
margin: 10px 20px;
margin: 10px 20px 30px;
margin: 10px 20px 30px 40px;

/* 单独设置 */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

/* 水平居中 */
.box {
  width: 200px;
  margin: 0 auto;             /* 上下0，左右自动（水平居中） */
}
注意： 垂直方向的margin会发生折叠（合并），取较大值；水平方向margin不会折叠。

🔹 四、CSS文本样式

控制文本的字体、大小、颜色、对齐方式等样式。

1. 字体样式

/* 字体相关 */
font-family: "Microsoft Yahei", "SimSun", sans-serif;  /* 字体族（备选机制） */
font-size: 16px;            /* 字体大小：px/em/rem/%等 */
font-weight: normal;        /* 字体粗细：normal(400)/bold(700)/100-900 */
font-style: normal;         /* 字体样式：normal/italic/oblique */
font-variant: normal;       /* 小型大写字母：normal/small-caps */

/* 字体简写（顺序：style variant weight size/line-height family） */
font: italic small-caps bold 16px/1.6 "Microsoft Yahei", sans-serif;

2. 文本颜色与对齐

/* 文本颜色 */
color: #333;                /* 十六进制 */
color: rgb(51, 51, 51);     /* RGB */
color: rgba(51, 51, 51, 0.8); /* RGBA（带透明度） */
color: hsl(0, 0%, 20%);     /* HSL */
color: hsla(0, 0%, 20%, 0.8); /* HSLA */
color: red;                 /* 颜色名（不推荐） */

text-align: center;         /* 居中对齐 */
text-align: right;          /* 右对齐 */
text-align: justify;        /* 两端对齐 */

/* 行高（行间距） */
line-height: normal;        /* 正常行高 */
line-height: 1.6;           /* 无单位（基于字体大小）- 推荐 */
line-height: 24px;          /* 固定值 */
line-height: 150%;          /* 百分比 */

3. 文本装饰与转换

/* 文本装饰 */
text-decoration: none;      /* 无装饰（常用於去除链接下划线） */
text-decoration: underline; /* 下划线 */

text-decoration-style: solid; /* 装饰线样式 */

/* 文本转换 */
text-transform: none;       /* 无转换 */

text-transform: lowercase;  /* 全部小写 */

/* 文本缩进 */
text-indent: 2em;           /* 首行缩进2个字符 */

/* 字母间距与单词间距 */

设置元素的背景颜色、背景图片、渐变效果等。

1. 背景基础属性

/* 背景颜色 */

/* 背景位置 */

🔹 六、CSS布局方式

CSS布局是前端开发的核心，常用的布局方式有：标准流、浮动布局、定位布局、Flex布局、Grid布局等。

1. 显示模式

/* 块级元素：独占一行，可设置宽高 */
display: block;             /* 块级 */
/* 行内元素：不独占一行，不可设置宽高 */
display: inline;            /* 行内 */
/* 行内块元素：不独占一行，可设置宽高 */
display: inline-block;      /* 行内块 */
/* 隐藏元素（不占空间） */
display: none;              /* 隐藏 */
/* Flex容器 */
display: flex;              /* 块级Flex */
display: inline-flex;       /* 行内Flex */
/* Grid容器 */
display: grid;              /* 块级Grid */
display: inline-grid;       /* 行内Grid */

/* 常见元素默认显示模式 */
/* 块级：div, p, h1-h6, ul, ol, li, dl, dt, dd, form, header, nav, main, section, footer等 */
/* 行内：span, a, em, strong, i, b, label等 */
/* 行内块：img, input, button, select等 */

2. 溢出处理

overflow: visible;         /* 可见（默认） */
overflow: hidden;          /* 隐藏溢出内容 */
overflow: scroll;          /* 显示滚动条（始终） */
overflow: auto;            /* 自动（需要时显示滚动条） */

/* 单独设置x/y轴 */
overflow-x: auto;
overflow-y: hidden;

3. 空白处理

white-space: normal;       /* 正常（默认） */

🔹 七、Flex弹性布局

Flex（Flexible Box）弹性布局，是目前最主流的布局方式，适用于一维布局（行或列）。

1. Flex容器属性（父元素）

/* 启用Flex布局 */
.container {
  display: flex;            /* 块级Flex */
  /* display: inline-flex; */ /* 行内Flex */
  
  /* 1. 主轴方向 */
  flex-direction: row;      /* 水平从左到右（默认） */
  flex-direction: row-reverse; /* 水平从右到左 */
  flex-direction: column;   /* 垂直从上到下 */
  flex-direction: column-reverse; /* 垂直从下到上 */
  
  /* 2. 换行方式 */
  flex-wrap: nowrap;        /* 不换行（默认） */
  flex-wrap: wrap;          /* 换行 */
  flex-wrap: wrap-reverse;  /* 反向换行 */
  
  /* 3. 方向+换行简写 */
  flex-flow: row wrap;      /* 默认：row nowrap */
  
  /* 4. 主轴对齐方式 */
  justify-content: flex-start; /* 左对齐（默认） */
  justify-content: flex-end;   /* 右对齐 */
  justify-content: center;     /* 居中 */
  justify-content: space-between; /* 两端对齐 */
  justify-content: space-around; /* 均匀分布（两侧有间距） */
  justify-content: space-evenly; /* 均匀分布（两侧无间距） */
  
  /* 5. 交叉轴对齐方式 */
  align-items: stretch;     /* 拉伸（默认） */
  align-items: flex-start;  /* 顶部对齐 */
  align-items: flex-end;    /* 底部对齐 */
  align-items: center;      /* 居中 */
  align-items: baseline;    /* 基线对齐 */
  
  /* 6. 多行交叉轴对齐（仅wrap时生效） */
  align-content: stretch;   /* 拉伸（默认） */
  align-content: flex-start;/* 顶部 */
  align-content: flex-end;  /* 底部 */
  align-content: center;    /* 居中 */
  align-content: space-between; /* 两端对齐 */
  align-content: space-around; /* 均匀分布 */
}

2. Flex项目属性（子元素）

/* Flex项目属性 */
.item {
  /* 1. 排序 */
  order: 0;                 /* 默认为0，数值越小越靠前 */
  
  /* 2. 放大比例 */
  flex-grow: 0;             /* 默认为0（不放大） */
  
  /* 3. 缩小比例 */
  flex-shrink: 1;           /* 默认为1（允许缩小） */
  
  /* 4. 基础尺寸 */
  flex-basis: auto;         /* 默认为auto（自身尺寸） */
  
  /* 5. 简写（grow shrink basis） */
  flex: 0 1 auto;           /* 默认值 */
  flex: 1;                  /* 等价于 1 1 0% */
  flex: auto;               /* 等价于 1 1 auto */
  flex: none;               /* 等价于 0 0 auto */
  
  /* 6. 单独对齐（覆盖align-items） */
  align-self: auto;         /* 继承父元素（默认） */
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: baseline;
  align-self: stretch;
}

3. Flex布局常用示例

1. 水平垂直居中： 
.container {
  display: flex;
  justify-content: center;  /* 水平居中 */
  align-items: center;      /* 垂直居中 */
  width: 300px;
  height: 200px;
  border: 1px solid #eee;
}

🔹 十一、CSS响应式设计

使网页能够适应不同屏幕尺寸的设备，提供良好的用户体验。

1. 视口设置

<!-- HTML头部必须设置 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

/* 解释：
   width=device-width - 视口宽度等于设备宽度
   initial-scale=1.0 - 初始缩放比例为1
   user-scalable=no - 禁止用户缩放（不推荐）
   maximum-scale=1.0 - 最大缩放比例
   minimum-scale=1.0 - 最小缩放比例
*/

2. 媒体查询（Media Query）

/* 基础语法 */
@media media-type and (media-feature) {
  /* CSS规则 */
}

/* 媒体类型 */
all - 所有设备（默认）
screen - 屏幕设备
print - 打印设备
speech - 语音设备

/* 媒体特性 */
width/height - 视口宽度/高度
min-width/max-width - 最小/最大宽度
min-height/max-height - 最小/最大高度
orientation - 方向（portrait/landscape）
aspect-ratio - 宽高比
device-width/device-height - 设备宽度/高度

/* 常用断点 */
/* 移动端 */
@media (max-width: 767px) {
  /* 手机样式 */
}

/* 平板 */
@media (min-width: 768px) and (max-width: 991px) {
  /* 平板样式 */
}

/* 桌面端 */
@media (min-width: 992px) and (max-width: 1199px) {
  /* 小屏桌面 */
}

/* 大屏桌面 */
@media (min-width: 1200px) {
  /* 大屏样式 */
}

/* 竖屏/横屏 */
@media (orientation: portrait) {
  /* 竖屏 */
}
@media (orientation: landscape) {
  /* 横屏 */
}

3. 响应式单位

单位 说明 使用场景
px 像素（固定单位） 边框、固定尺寸
% 百分比（相对父元素） 宽度、高度
em 相对当前元素字体大小 字体大小、内边距
rem 相对根元素（html）字体大小 响应式布局（推荐）
vw 视口宽度的1%（Viewport Width） 响应式宽度、字体
vh 视口高度的1%（Viewport Height） 响应式高度
vmin vw和vh中的较小值 适配不同方向
vmax vw和vh中的较大值 适配不同方向
ch 字符宽度（0的宽度） 文本宽度限制

响应式设计原则：
1. 移动优先（Mobile First）：先设计移动端，再扩展到桌面端
