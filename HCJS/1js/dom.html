一、DOM基础认知 (从零开始)

1.1 什么是DOM？

DOM (Document Object Model) 即文档对象模型，是W3C组织推荐的处理可扩展标记语言的标准编程接口。简单来说：

  • DOM把HTML/XML文档转换成一个树形结构
  • 每个节点都是一个对象，我们可以通过JS操作这些对象
  • 通过DOM，我们可以修改页面的内容、结构和样式

1.2 DOM树结构

一个简单的HTML文档对应的DOM树：


document (文档节点)
└── html (元素节点)
    ├── head (元素节点)
    │   └── title (元素节点)
    │       └── "页面标题" (文本节点)
    └── body (元素节点)
        ├── h1 (元素节点)
        │   └── "Hello World" (文本节点)
        └── p (元素节点)
            └── "这是一段文字" (文本节点)
            └── <a> (元素节点)
                └── "链接" (文本节点)

1.3 获取DOM元素 (基础操作)

这是DOM操作的第一步，先找到要操作的元素，常用方法如下：

方法 说明 返回值
getElementById() 通过ID获取元素 单个元素对象
getElementsByTagName() 通过标签名获取元素 HTMLCollection集合
getElementsByClassName() 通过类名获取元素 HTMLCollection集合
querySelector() 通过选择器获取第一个匹配元素 单个元素对象
querySelectorAll() 通过选择器获取所有匹配元素 NodeList集合

代码示例：


// 1. 通过ID获取元素 (最常用，ID唯一)
const box = document.getElementById('box');

// 2. 通过标签名获取 (返回集合，需遍历)
const lis = document.getElementsByTagName('li');
for(let i = 0; i < lis.length; i++) {
    console.log(lis[i]);
}

// 3. 通过类名获取 (返回集合)
const items = document.getElementsByClassName('item');

// 4. 通过选择器获取 (现代开发首选，灵活强大)
const firstItem = document.querySelector('.item'); // 获取第一个类为item的元素
const allItems = document.querySelectorAll('.item'); // 获取所有类为item的元素

// 5. 获取特殊元素
const html = document.documentElement; // 获取html根元素
const body = document.body; // 获取body元素

二、DOM进阶操作 (属性、内容、样式)

2.1 操作元素内容

  • innerText：获取/设置元素的文本内容（不解析HTML）
  • innerHTML：获取/设置元素的HTML内容（解析HTML标签）
  • textContent：获取所有文本内容（包括隐藏的）

// 获取元素
const contentBox = document.getElementById('contentBox');

// 1. innerText - 只处理文本
console.log(contentBox.innerText); // 获取文本
contentBox.innerText = '新的文本内容'; // 设置文本

// 2. innerHTML - 处理HTML
console.log(contentBox.innerHTML); // 获取HTML内容
contentBox.innerHTML = '<h3>这是新的HTML内容</h3>'; // 设置HTML

// 3. textContent
console.log(contentBox.textContent); // 获取所有文本

2.2 操作元素属性

分为内置属性（如src、href、id）和自定义属性：


// 获取元素
const img = document.querySelector('img');

// 1. 操作内置属性
console.log(img.src); // 获取src属性
img.alt = '风景图片'; // 设置alt属性

// 2. 操作自定义属性 (推荐使用dataset)
console.log(img.dataset.index); // 获取 data-index 属性
img.dataset.info = '自定义信息'; // 设置 data-info 属性

// 3. 通用方法
img.setAttribute('title', '图片标题'); // 设置属性
const title = img.getAttribute('title'); // 获取属性
img.removeAttribute('title'); // 删除属性

2.3 操作元素样式

分为行内样式和类样式（推荐类样式）：


// 获取元素
const styleBox = document.getElementById('styleBox');

// 1. 行内样式操作
styleBox.style.width = '200px'; // 设置宽度
styleBox.style.height = '100px'; // 设置高度
styleBox.style.backgroundColor = '#4299e1'; // 设置背景色
styleBox.style.cssText = 'width:200px; height:100px; color:white;'; // 批量设置

// 2. 获取计算样式 (实际渲染的样式)
const computedStyle = window.getComputedStyle(styleBox);
console.log(computedStyle.width); // 获取计算后的宽度

// 3. 类样式操作 (推荐！结构和样式分离)
styleBox.classList.add('active'); // 添加类
styleBox.classList.remove('active'); // 移除类
styleBox.classList.toggle('active'); // 切换类 (有则删，无则加)
styleBox.classList.contains('active'); // 判断是否有该类

三、DOM节点操作 (创建、添加、删除)

3.1 节点的分类和属性

  • 元素节点：nodeType = 1 (如div、p、a)
  • 文本节点：nodeType = 3 (文字内容)
  • 属性节点：nodeType = 2 (元素的属性)

3.2 创建和添加节点


// 1. 创建元素节点
const newDiv = document.createElement('div');
newDiv.className = 'new-item'; // 设置类名
newDiv.innerText = '这是新创建的元素'; // 设置内容

// 2. 创建文本节点 (一般用innerText/innerHTML替代)
const textNode = document.createTextNode('这是文本节点');

// 3. 添加节点到文档中
const parent = document.getElementById('parentBox');
parent.appendChild(newDiv); // 添加到父元素末尾
// parent.insertBefore(newDiv, parent.firstChild); // 添加到指定位置前

// 4. 克隆节点
const cloneDiv = newDiv.cloneNode(true); // true表示深克隆(包含子节点)
parent.appendChild(cloneDiv);

3.3 删除和替换节点


// 获取元素
const parent = document.getElementById('parentBox');
const removeItem = document.querySelector('.new-item');

// 1. 删除节点
if(removeItem) {
    parent.removeChild(removeItem); // 父元素删除子节点
    // 或 removeItem.remove(); // 直接删除自己 (现代浏览器支持)
}

// 2. 替换节点
const newNode = document.createElement('p');
newNode.innerText = '替换后的节点';
if(removeItem) {
    parent.replaceChild(newNode, removeItem);
}

四、DOM实战案例 (综合应用)

4.1 案例1：待办事项列表 (ToDo List)

功能说明：添加待办、删除待办、标记完成


// 待办事项案例核心代码
function addTodo() {
    // 1. 获取输入框内容
    const input = document.getElementById('todoInput');
    const todoText = input.value.trim();
    
    // 2. 验证内容
    if(!todoText) {
        alert('请输入待办事项！');
        return;
    }
    
    // 3. 创建待办项元素
    const todoItem = document.createElement('div');
    todoItem.style.display = 'flex';
    todoItem.style.alignItems = 'center';
    todoItem.style.margin = '8px 0';
    todoItem.style.padding = '8px';
    todoItem.style.backgroundColor = '#f8f9fa';
    todoItem.style.borderRadius = '4px';
    
    // 4. 创建复选框
    const checkbox = document.createElement('input');
    checkbox.type = 'checkbox';
    checkbox.onchange = function() {
        // 切换完成状态
        textSpan.style.textDecoration = this.checked ? 'line-through' : 'none';
        textSpan.style.color = this.checked ? '#999' : '#333';
    };
    
    // 5. 创建文本节点
    const textSpan = document.createElement('span');
    textSpan.style.margin = '0 10px';
    textSpan.innerText = todoText;
    
    // 6. 创建删除按钮
    const deleteBtn = document.createElement('button');
    deleteBtn.innerText = '删除';
    deleteBtn.style.padding = '4px 8px';
    deleteBtn.style.fontSize = '0.8rem';
    deleteBtn.onclick = function() {
        todoItem.remove(); // 删除当前待办项
    };
    
    // 7. 组装并添加到页面
    todoItem.appendChild(checkbox);
    todoItem.appendChild(textSpan);
    todoItem.appendChild(deleteBtn);
    document.getElementById('todoList').appendChild(todoItem);
    
    // 8. 清空输入框
    input.value = '';
    input.focus();
}

// 图片轮播核心代码
let currentIndex = 0;
const slides = document.getElementById('carouselInner').children;
const totalSlides = slides.length;
const indicators = document.getElementById('indicators').children;

// 更新指示器样式
function updateIndicators() {
    for(let i = 0; i < indicators.length; i++) {
        indicators[i].style.background = i === currentIndex ? '#4299e1' : '#ccc';
    }
}

// 切换到指定幻灯片
function goToSlide(index) {
    currentIndex = index;
    // 计算偏移量
    const carouselInner = document.getElementById('carouselInner');
    carouselInner.style.transform = `translateX(-${currentIndex * 100}%)`;
    updateIndicators();
}

// 上一张
function prevSlide() {
    currentIndex = (currentIndex - 1 + totalSlides) % totalSlides;
    goToSlide(currentIndex);
}

// 下一张
function nextSlide() {
    currentIndex = (currentIndex + 1) % totalSlides;
    goToSlide(currentIndex);
}

// 自动轮播
let carouselTimer = setInterval(nextSlide, 3000);

// 鼠标悬停暂停轮播
document.getElementById('carousel').onmouseenter = function() {
    clearInterval(carouselTimer);
};

// 鼠标离开继续轮播
document.getElementById('carousel').onmouseenter = function() {
    carouselTimer = setInterval(nextSlide, 3000);
};

// 初始化
updateIndicators();

五、DOM操作注意事项和最佳实践

5.1 性能优化

  • 减少DOM查询：将查询结果缓存到变量中，避免重复查询
  • 批量操作DOM：使用DocumentFragment或隐藏容器批量添加节点
  • 减少重排重绘：先隐藏元素再修改，修改完成后显示
  • 使用事件委托：利用事件冒泡，给父元素绑定事件处理子元素

5.2 常见坑点

  • 获取元素时确保DOM已加载完成 (放在DOMContentLoaded事件中)
  • 区分HTMLCollection和NodeList，前者是动态集合，后者是静态
  • 操作样式时优先使用class，避免大量行内样式
  • 删除节点前先判断节点是否存在，避免报错

5.3 现代DOM操作趋势

  • 使用querySelector/querySelectorAll替代旧方法
  • 使用dataset操作自定义属性
  • 使用classList操作类名
  • 结合事件委托提高性能
