# server.js 涉及的核心知识体系（层级标题版）
## 一、Node.js 基础核心
1. Node.js 运行环境与服务端执行机制
2. CommonJS 模块规范（require 引入模块、module.exports 导出模块）
3. Node.js 全局内置变量（__dirname、__filename）
4. Node.js 命令行运行与项目启动

## 二、Express 框架核心应用
1. Express 实例创建与 HTTP 服务启动（app = express()、app.listen()）
2. Express 路由（接口）定义（POST/GET 请求、路由路径匹配）
3. Express 中间件机制（全局中间件、路由级中间件、自定义中间件）
4. Express 请求对象（req）与响应对象（res）操作
5. Express 路由与中间件的执行顺序

## 三、Node.js 内置核心模块
1. fs 模块（文件系统操作）- 同步/异步文件/文件夹处理
2. path 模块（路径处理）- 跨平台路径拼接、路径解析
3. timers 定时器模块（setTimeout 延迟执行）
4. http 模块（底层网络支持，Express 封装基于此）

## 四、第三方依赖模块应用
1. body-parser - HTTP 请求体解析（JSON/URL 编码格式）
2. cookie-parser - Cookie 解析与读取
3. morgan - HTTP 请求日志记录
4. jsonwebtoken（JWT）- 令牌生成与验证
5. xlsx - JSON 数据转 Excel 文件、Excel 生成与写入
6. mysql2（或其他数据库模块）- 关系型数据库连接与操作
7. express-static - 静态资源托管（Express 内置封装）

## 五、认证与权限控制
1. JWT 认证机制（令牌生成、令牌验证、令牌有效期）
2. Cookie 操作（服务端设置 Cookie、Cookie 有效期与域名绑定）
3. 自定义认证中间件 - 接口权限校验
4. 请求头解析（Authorization 头提取 Token）
5. 身份双重验证（Cookie + Token 一致性校验）

## 六、数据库操作（服务端核心）
1. 关系型数据库（MySQL）连接池配置与使用
2. SQL 语句执行（SELECT 多表查询、条件查询）
3. 防 SQL 注入（占位符 ? 传参）
4. 异步数据库操作与结果处理

## 七、文件系统与文件操作
1. 文件夹创建（判断文件夹是否存在、递归/非递归创建）
2. 文件写入（普通写入、流式写入 createWriteStream）
3. 文件读取与删除（同步删除 unlinkSync、文件存在性判断）
4. 临时文件管理（文件命名、文件存储路径规划）

## 八、HTTP 网络与接口开发
1. HTTP 常用请求方法（POST/GET/OPTIONS）
2. HTTP 请求头/请求体/Cookie 解析与处理
3. HTTP 响应状态码设置（200/401/500/404）
4. JSON 格式统一响应（res.json()）
5. CORS 跨域问题解决（自定义跨域中间件、跨域响应头设置）
6. 预检请求（OPTIONS）处理

## 九、异步编程
1. Promise 异步编程模型
2. async/await 异步语法（异步操作同步化、异步函数定义）
3. 异步错误捕获与处理
4. 定时器异步任务（setTimeout 实现延迟操作）
5. 异步文件操作、异步数据库操作

## 十、错误处理与日志管理
1. 局部错误捕获（try/catch 捕获异步错误）
2. Express 全局错误中间件（兜底处理未捕获错误）
3. 分级日志记录（访问日志、错误日志）
4. 日志持久化（日志写入本地文件）
5. 错误信息格式化与关键信息记录（请求IP、接口路径、错误栈）
6. 统一错误响应格式（前端友好的错误提示）

## 十一、静态资源与文件导出/下载
1. Express 静态资源托管中间件（express.static）
2. Excel 文件生成与导出（JSON 转 Excel 工作表/工作簿）
3. 服务端文件下载链接生成
4. 静态资源访问路径配置

## 十二、接口开发规范
1. 接口参数接收与校验（req.body 获取前端传参）
2. 受保护接口（需要登录的接口）权限控制
3. 接口返回值统一格式（success、data、message 字段）
4. 大请求体限制配置（请求体大小限制）

## 十三、配置与常量管理
1. 服务端自定义常量配置（JWT 密钥、Token 有效期、前端域名）
2. 端口号、数据库连接配置、文件存储路径配置
3. 配置项统一管理与维护

## 十四、服务端性能与资源管理
1. 临时文件定时清理（防止磁盘资源占用）
2. 数据库连接池（连接复用，提升数据库操作效率）
3. 流式写入（大文件/日志写入，提升性能）
