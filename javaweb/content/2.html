2.1 HTML 核心标签与语义化（基础结构）

2.1.1 HTML5 文档基本结构

HTML5是当前主流版本，支持语义化标签，适配现代浏览器，以下是标准文档结构（可直接在IDEA2025中新建HTML文件生成）：

<!DOCTYPE html>  
<html lang="zh-CN">  
<head>
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0">  
    <title>页面标题</title>  
    <link rel="stylesheet" href="style.css">  
</head>
<body>
    
    <script src="script.js"></script>  
</body>
</html>
💡 IDEA2025快捷技巧：新建HTML文件后，输入! + 回车，自动生成HTML5标准结构；输入标签名 + tab，自动补全标签（如输入div+tab生成<div></div>）。

2.1.3 表单标签（JavaWeb核心交互入口）

表单是客户端向服务器提交数据的核心方式，以下是完整的表单结构（后续会用CSS美化）：

<form action="/login" method="post">
    <div class="form-group">
        <label for="username">用户名：</label>
        <input type="text" id="username" name="username" required placeholder="请输入用户名">
    </div>
    <div class="form-group">
        <label for="password">密码：</label>
        <input type="password" id="password" name="password" required placeholder="请输入密码">
    </div>
    <div class="form-group">
        <label for="gender">性别：</label>
        <input type="radio" id="male" name="gender" value="male" checked> 男
        <input type="radio" id="female" name="gender" value="female"> 女
    </div>
    <div class="form-group">
        <label for="hobby">爱好：</label>
        <input type="checkbox" name="hobby" value="code"> 编程
        <input type="checkbox" name="hobby" value="read"> 阅读
    </div>
    <div class="form-group">
        <label for="city">城市：</label>
        <select name="city" id="city">
            <option value="beijing">北京</option>
            <option value="shanghai" selected>上海</option>
            <option value="guangzhou">广州</option>
        </select>
    </div>
    <button type="submit">提交</button>
    <button type="reset">重置</button>
</form>
❗ 关键属性说明： 1. action：表单提交的服务器地址（后续Servlet接收地址）； 2. method：提交方式（GET/POST，JavaWeb中优先用POST，更安全）； 3. name：服务器获取参数的标识（必须设置，和后端参数名对应）； 4. required：HTML5新增，必填项验证。

2.2 CSS 核心与美化实战（视觉层）

2.2.1 CSS 引入方式

三种引入方式对比
引入方式 语法 适用场景
行内样式 <div style="color:red;"></div> 临时样式、单个元素
内部样式 <style>div{color:red;}</style> 单页面样式、小项目
外部样式 <link rel="stylesheet" href="style.css"> 多页面共享（推荐，如你的style.css）

2.2.2 CSS 核心选择器（精准定位元素）

/* 1. 元素选择器：选中所有指定标签 */
body { font-family: "Microsoft YaHei", sans-serif; }

/* 2. 类选择器（最常用）：选中class="form-group"的元素 */
.form-group { margin-bottom: 20px; }

/* 3. ID选择器：选中id="username"的元素（唯一） */
#username { width: 100%; }

/* 4. 后代选择器：选中.form-group下的所有label */
.form-group label { display: block; }

/* 5. 伪类选择器：交互状态（核心美化） */
input:focus { border-color: #3b82f6; }  /* 输入框聚焦 */
button:hover { background: #1e40af; }  /* 按钮悬浮 */

/* 6. 属性选择器：选中type="password"的input */
input[type="password"] { font-family: monospace; }

2.2.3 盒模型（布局核心）

所有HTML元素都是「盒子」，由content（内容）、padding（内边距）、border（边框）、margin（外边距）组成：

/* 标准盒模型（推荐）：width/height仅包含content */
box-sizing: content-box;

/* 边框盒模型（实战常用）：width/height包含content+padding+border */
box-sizing: border-box;

/* 通用重置（所有元素用边框盒模型） */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2.3.3 事件处理（交互核心）

事件是用户操作（点击、输入、鼠标移动）的响应，以下是表单交互示例：

// 方式1：直接绑定（简单）
document.getElementById("loginBtn").onclick = function() {
    const username = document.getElementById("username").value;
    const password = document.getElementById("password").value;
    
    // 表单验证
    if (!username) {
        alert("请输入用户名");
        return;
    }
    if (password.length < 6) {
        alert("密码长度不能少于6位");
        return;
    }
    
    // 提交表单（或发送AJAX请求）
    alert(`登录成功！欢迎 ${username}`);
};

// 方式2：事件监听（推荐，可绑定多个事件）
const loginForm = document.querySelector("form");
loginForm.addEventListener("submit", function(e) {
    e.preventDefault(); // 阻止表单默认提交（核心，用于AJAX提交）
    
    // 获取所有表单数据
    const formData = new FormData(loginForm);
    const username = formData.get("username");
    const password = formData.get("password");
    
    // 验证逻辑...
    console.log("表单数据：", {username, password});
});

// 常用事件
// 输入框输入事件（实时验证）
usernameInput.addEventListener("input", function() {
    if (this.value.length > 10) {
        this.classList.add("warning");
    } else {
        this.classList.remove("warning");
    }
});

// 鼠标悬浮事件
const btn = document.querySelector(".btn-primary");
btn.addEventListener("mouseenter", function() {
    this.style.transform = "scale(1.05)"; // 放大
});
btn.addEventListener("mouseleave", function() {
    this.style.transform = "scale(1)"; // 还原
});

2.3.4 AJAX 异步请求（JavaWeb核心）

AJAX（异步JavaScript和XML）是前端不刷新页面向服务器请求数据的核心方式，以下是Fetch API示例（ES6+）：

// 发送POST请求（登录示例）
document.querySelector("form").addEventListener("submit", async function(e) {
    e.preventDefault();
    
    const formData = {
        username: document.getElementById("username").value,
        password: document.getElementById("password").value
    };
    
    try {
        // 发送请求到后端Servlet
        const response = await fetch("/login", {
            method: "POST",
            headers: {
                "Content-Type": "application/json"
            },
            body: JSON.stringify(formData)
        });
        
        // 处理响应
        const result = await response.json();
        if (result.success) {
            alert("登录成功！");
            window.location.href = "/home"; // 跳转到首页
        } else {
            alert("登录失败：" + result.message);
        }
    } catch (error) {
        console.error("请求失败：", error);
        alert("网络错误，请重试");
    }
});
💡 注意：AJAX请求需要后端Servlet支持JSON解析和响应，后续章节会详细讲解Servlet处理AJAX请求的方法。
