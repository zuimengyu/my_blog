第6章：文件上传与下载（小白完整版·Tomcat11）

6.0 前置核心：文件上传下载是啥？

大白话：
👉 上传：把本地文件（图片/文档）传到服务器保存
👉 下载：把服务器里的文件拿到本地

💡 常用场景：
📌 上传：头像、身份证照片、Excel报表
📌 下载：报表、模板、用户上传的文件

核心前提：
1. 前端表单必须用 enctype="multipart/form-data"（二进制传输）
2. 后端用 commons-fileupload 工具（不用自己写二进制解析）

6.1 Maven 依赖（文件上传核心）

文件上传下载必备依赖（直接复制） 
<!-- 文件上传核心依赖 -->
<dependency>
    <groupId>commons-fileupload</groupId>
    <artifactId>commons-fileupload</artifactId>
    <version>1.5</version>
</dependency>

<!-- 依赖的IO工具包 -->
<dependency>
    <groupId>commons-io</groupId>
    <artifactId>commons-io</artifactId>
    <version>2.15.1</version>
</dependency>

<!-- Servlet依赖（已加，确保是jakarta包） -->
<dependency>
    <groupId>jakarta.servlet</groupId>
    <artifactId>jakarta.servlet-api</artifactId>
    <version>6.0.0</version>
    <scope>provided</scope>
</dependency>
❗ 小白必坑：
1. 必须加 commons-io，否则文件上传报错
2. 版本用最新稳定版，避免兼容性问题

6.2 实战1：单文件上传（头像/文档）

6.2.1 前端上传页面（upload.html）

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>文件上传</title>
</head>
<body>
    <h3>文件上传</h3>
    <!-- 核心：method=post + enctype=multipart/form-data -->
    <form action="/uploadServlet" method="post" enctype="multipart/form-data">
        选择文件：<input type="file" name="file"><br>
        备注：<input type="text" name="desc"><br>
        <input type="submit" value="上传">
    </form>
</body>
</html>
❗ 核心必写：
1. 表单 method 必须是 post（get 传不了大文件）
2. enctype 必须是 multipart/form-data（二进制传输）

6.2.2 后端上传Servlet（UploadServlet.java）

package com.servlet;

import org.apache.commons.fileupload.FileItem;
import org.apache.commons.fileupload.FileUploadException;
import org.apache.commons.fileupload.disk.DiskFileItemFactory;
import org.apache.commons.fileupload.servlet.ServletFileUpload;
import jakarta.servlet.annotation.WebServlet;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import java.io.File;
import java.io.IOException;
import java.util.List;

@WebServlet("/uploadServlet")
public class UploadServlet extends HttpServlet {
    @Override
    protected void doPost(HttpServletRequest request, HttpServletResponse response) throws IOException {
        // 1. 设置编码
        request.setCharacterEncoding("UTF-8");
        response.setContentType("text/html;charset=UTF-8");

        // 2. 判断是否是多段式上传（防止普通表单提交）
        if (!ServletFileUpload.isMultipartContent(request)) {
            response.getWriter().write("不是文件上传表单！");
            return;
        }

        try {
            // 3. 创建文件上传工厂
            DiskFileItemFactory factory = new DiskFileItemFactory();
            // 设置临时文件目录（文件过大时先存临时文件）
            String tempPath = getServletContext().getRealPath("/temp");
            factory.setRepository(new File(tempPath));

            // 4. 创建文件上传核心对象
            ServletFileUpload upload = new ServletFileUpload(factory);
            // 设置文件大小限制：10MB
            upload.setFileSizeMax(10 * 1024 * 1024);

            // 5. 解析请求，获取所有表单项
            List<FileItem> items = upload.parseRequest(request);

            // 6. 遍历表单项（区分普通字段和文件字段）
            for (FileItem item : items) {
                // 普通表单字段（如备注）
                if (item.isFormField()) {
                    String fieldName = item.getFieldName(); // 字段名
                    String value = item.getString("UTF-8"); // 字段值（解决中文乱码）
                    System.out.println("备注：" + value);
                } 
                // 文件字段（要上传的文件）
                else {
                    // 获取文件名（处理浏览器兼容，只取最后一个/后的名字）
                    String fileName = item.getName();
                    fileName = fileName.substring(fileName.lastIndexOf("\\") + 1);
                    System.out.println("上传文件名：" + fileName);

                    // 7. 定义文件保存路径（服务器webapp/upload目录）
                    String savePath = getServletContext().getRealPath("/upload");
                    File saveFile = new File(savePath, fileName);

                    // 8. 确保目录存在（不存在则创建）
                    if (!saveFile.getParentFile().exists()) {
                        saveFile.getParentFile().mkdirs();
                    }

                    // 9. 写入文件到服务器
                    item.write(saveFile);
                    response.getWriter().write("文件上传成功！路径：" + saveFile.getAbsolutePath());

                    // 10. 删除临时文件（可选）
                    item.delete();
                }
            }
        } catch (FileUploadException e) {
            response.getWriter().write("文件上传失败：" + e.getMessage());
            e.printStackTrace();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
💡 小白必改：
1. 文件大小限制可调整（10*1024*1024=10MB）
2. 保存路径可自定义（/upload 是项目根目录下的文件夹）

6.3 实战2：文件下载（服务器文件到本地）

6.3.1 前端下载页面（download.html）

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>文件下载</title>
</head>
<body>
    <h3>文件下载</h3>
    <a href="/downloadServlet?fileName=test.jpg">下载test.jpg</a><br>
    <a href="/downloadServlet?fileName=文档.txt">下载文档.txt</a>
</body>
</html>

6.3.2 后端下载Servlet（DownloadServlet.java）

package com.servlet;

import jakarta.servlet.ServletContext;
import jakarta.servlet.ServletOutputStream;
import jakarta.servlet.annotation.WebServlet;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import java.io.FileInputStream;
import java.io.IOException;
import java.net.URLEncoder;

@WebServlet("/downloadServlet")
public class DownloadServlet extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws IOException {
        // 1. 获取要下载的文件名（解决中文参数乱码）
        String fileName = request.getParameter("fileName");
        fileName = new String(fileName.getBytes("ISO-8859-1"), "UTF-8");

        // 2. 获取文件在服务器的路径（webapp/upload目录）
        ServletContext context = getServletContext();
        String filePath = context.getRealPath("/upload/" + fileName);
        System.out.println("文件路径：" + filePath);

        // 3. 设置响应头（告诉浏览器是下载文件）
        // 3.1 设置文件类型
        String mimeType = context.getMimeType(fileName);
        response.setContentType(mimeType);
        // 3.2 设置下载弹窗（解决中文文件名乱码）
        response.setHeader("Content-Disposition", "attachment;filename=" + URLEncoder.encode(fileName, "UTF-8"));

        // 4. 读取服务器文件，写入响应流
        FileInputStream fis = new FileInputStream(filePath);
        ServletOutputStream sos = response.getOutputStream();

        // 5. 缓冲区读写
        byte[] buffer = new byte[1024];
        int len;
        while ((len = fis.read(buffer)) != -1) {
            sos.write(buffer, 0, len);
        }

        // 6. 关闭流
        sos.close();
        fis.close();
    }
}
❗ 小白必坑：
1. 中文文件名必须用 URLEncoder.encode 编码，否则下载文件名乱码
2. 确保服务器/upload目录下有对应文件，否则报文件找不到

6.4 本章终极总结（小白记住这些就够）

  • 🍉 上传核心：表单 enctype="multipart/form-data" + commons-fileupload 解析
  • 🔐 下载核心：设置响应头 Content-Disposition: attachment + 中文文件名URLEncoder编码
  • ⚡ 依赖必加：commons-fileupload + commons-io
  • 🎯 坑点：上传用POST、设置文件大小限制、确保服务器目录存在
✅ 已删除：多文件上传（单文件会了，多文件只需要循环）、文件分片上传（进阶）、复杂文件验证（小白不用）
